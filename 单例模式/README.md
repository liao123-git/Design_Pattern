# 单例模式
### 介绍
- 系统中被唯一使用
- 一个类只有一个实例
- 比如说登录框、购物车，都只有一个

### 场景
- jQuery只有一个$，不管引入了多少个jQuery都只有一个$
- 不管获取了多少个指向的都是同一个实例，数据都是共享的

### 设计原则验证
- 符合单一职责原则，只实例化唯一的对象
- 没法具体开放封闭原则，但是绝对不违反开放封闭原则
